# 时间任务提取与日历提醒助手 - 打包说明

这是当前网站项目的可部署源码包。

## 本地运行

先安装依赖：

```bash
npm install
```

再启动网站：

```bash
npm start
```

浏览器打开：

```text
http://127.0.0.1:3000
```

## 部署到免费公网

推荐使用 Render 免费版。

1. 把本项目上传到 GitHub。
2. 打开 `https://render.com`。
3. 选择 `New` -> `Blueprint`。
4. 连接 GitHub 仓库。
5. 选择这个项目仓库。
6. Render 会自动读取 `render.yaml`。
7. 部署完成后，Render 会生成一个 `onrender.com` 公网链接。

## 当前主要功能

- 上传 Word、PDF、Excel、图片，或直接粘贴文字。
- 自动提取时间安排。
- Excel 目前只识别 `巡检时间` 和 `维修时间` 两类列。
- 可设置统一提前提醒时间，也可给每条日程单独设置。
- 生成 `.ics` 日历文件，导入手机日历后由手机系统提醒。

## 注意事项

- 免费公网平台可能会休眠，第一次打开可能较慢。
- 当前版本使用日历提醒，不是安卓系统闹钟。
- 手机导入日历是否弹出导入界面，取决于手机浏览器和日历 App。
- 如果部署后 `/healthz` 可以打开，说明后端服务运行正常。
