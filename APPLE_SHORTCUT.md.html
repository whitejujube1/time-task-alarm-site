# iPhone 快捷指令导入提醒事项

网站会生成一个 JSON 数据链接，并尝试打开名为 `导入提醒事项` 的快捷指令。

## 快捷指令名称

必须叫：

```text
导入提醒事项
```

## 快捷指令动作思路

1. 接收快捷指令输入。
2. 获取 URL 内容。
3. 从 JSON 中取 `reminders`。
4. 对每一项重复：
   - 标题：`title`
   - 备注：`notes`
   - 到期日期：`dueDate`
   - 提醒时间：`remindAt`
   - 添加新的提醒事项。

## 网站按钮

识别完成后点击：

```text
2. 运行快捷指令导入
```

如果快捷指令没有自动打开，说明 iPhone 上还没有创建同名快捷指令，或浏览器拦截了 `shortcuts://` 跳转。
